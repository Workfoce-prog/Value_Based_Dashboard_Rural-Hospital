
# Value-Based Care Readiness Dashboard

This Streamlit dashboard allows rural hospitals to:
- Enter and score key metrics
- Visualize RAG status
- Download results
- Use built-in sample data for testing

## Sample Variables
- Readmission Rate
- Chronic Disease Score
- Patient Engagement
- Population Health Index
- Timely Follow-Up
- Social Determinants Screening
- Preventive Care Utilization

To run the app:
1. Upload a CSV using the sample structure
2. Use the sidebar to browse and compare hospitals
3. Visualizations will update dynamically
